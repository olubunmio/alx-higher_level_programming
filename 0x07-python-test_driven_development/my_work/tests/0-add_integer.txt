===============================================
Using add_integer() function
===============================================

Examples:
    >>> add_integer = __import__("0-add_integer").add_integer
    >>> add_integer(1, 2)
    3
    >>> add_integer(100, -2)
    98
    >>> add_integer(2)
    100
    >>> add_integer(100.3, -2)
    98

``a`` and ``b`` must be integers or floats, otherwise raise a TypeError
exception with the message ``a must be an integer`` or ``b must be an
integer``
